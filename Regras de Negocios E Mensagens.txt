#- REGRAS DE NEGÓCIOS -#
RN-01 - Para efetuar login, o Usuário precisa possuir as credenciais que foram disponibilizadas pela academia.

RN-02 - O login só é validado quando o os campos e-mail e senha corresponderem aos dados preenchidos na tabela BULLK_USUARIO. {RN-01} 

RN-03 - Os usuários cadastrados com Tipo de Usuário “Aluno”, apenas conseguirão realizar o login na aplicação App [IMG-01], enquanto os usuários cadastrados com Tipo de Usuário “Admin”, apenas conseguirão realizar o login na aplicação Web [IMG-03]. {RN-02}

RN-04 - O Usuário que realiza o cadastro ou as alterações deve estar registrado com o Tipo de Usuário como “Admin”.

RN-05 - Para que cadastro seja realizado com sucesso, todos os campos obrigatórios, sinalizados com “*”, devem ser preenchidos.

RN-06 - Um Usuário não pode ser cadastrado com o mesmo e-mail de outro Usuário.

RN-07 - Os dados informados nos campos devem ser válidos.
			- Nome: tamanho não pode ser maior que 60 caracteres;
			- Data Nascimento: não pode ser maior ou igual a hoje;
			- Celular: deve conter apenas números, e deve conter menos que 15 caracteres;
			- E-mail: deve conter um e-mail válido;
			- UrlAvatar: deve conter um url válido;
			- Senha: deve conter mais de 8 caracteres.
			
RN-08 - Para alteração de senhas, o usuário terá que entrar em contato com a Secretária da Academia;
			
RN-09 - Os dados informados nos campos devem ser válidos.
			- Descrição: tamanho não pode ser maior que 60 caracteres;
			- Arquivo Avaliação: deve conter um arquivo tipo pdf válido.
			
RN-10 - Só poderá inserir uma avaliação para usuários que estão ativos.
			
RN-11 - O usuário deve permitir que o aplicativo realize downloads para o dispositivo.

RN-12 - Os dados informados nos campos devem ser válidos.

RN-13 - O Aluno consegue visualizar apenas os treinos atribuídos ao seu usuário, e que possuem status cadastrado como “Ativo”.
RN-14 - Para que um treino possa ser considerado como alternativo, as seguintes condições devem ser satisfeitas:
			- O Exercício não pode estar cadastrado como ativo no conjunto de treinos do Aluno
			- O “Grupo Músculos” cadastrado no exercício deve ser o mesmo

RN-15 - Não é possível fazer o processo de Inativação para Exercícios que estejam vinculados a um Treino ativo;

RN-16 - Os dados informados nos campos devem ser válidos.
			- Descrição: tamanho não pode ser maior que 60 caracteres;
			- Imagem Ilustração: deve conter um arquivo do tipo imagem válido;
			- Vídeo Ilustração: deve conter um url válido.
			
RN-17 - Os dados informados nos campos devem ser válidos.
			- Descrição: tamanho não pode ser maior que 60 caracteres;
			
RN-18 - Não é possível fazer o processo de Inativação para Aparelhos que estejam vinculados a um Treino;
